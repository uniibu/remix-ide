Circom Compiler
=============

.. meta::
   :description: Compile Noir zero-knowledge circuits in Remix IDE, generate ZK proofs, and verify them using Solidity smart contracts.
   :keywords: noir compiler, zero knowledge, zk proof, remix ide, noir zkp

Circom is a low level language for writing cryptographic circuits. This guide covers the entire workflow of using the Circom Compiler plugin in Remix IDE to compile Circom code, generate a zero-knowledge proof, and verify the generated proof using a Solidity smart contract and a js script.
